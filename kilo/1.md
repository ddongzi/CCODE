\x1b, Escape character (转义字符27)

Escape sequence such as coloring text, moving the cursor around, and clearing parts of the screen.

J 代表 清除 命令
2 为 命令参数。  含义为清除screen， 还有0，1..

[所有的转义序列](https://vt100.net/docs/vt100-ug/chapter3.html)

<esc>[1;1H  光标回到左上角
<esc>[H 光标回到左上角

C command (Cursor Forward) moves the cursor to the right
B command (Cursor Down) moves the cursor down. 
[999C 参数为移动距离，999保证移动到边

 <esc>[999;999H 不能使用， 不能保证会发生什么

 上箭头键（↑）：发送的是 ESC (ASCII 27) + [A，而不是 CTRL+A。
